## Описание проекта

Проект представляет собой консольную программу на Python для анализа погодных данных за год.

Программа работает с CSV-файлом, в котором для каждого дня указана температура. Пользователь может получить среднюю температуру по месяцам, найти самый тёплый и самый холодный день, отсортировать месяцы по температуре, найти дни теплее заданного значения, выполнить сглаживание температурного ряда и отменить последнее преобразование данных.

В проекте используются следующие алгоритмы и структуры данных:

* префиксные суммы
* линейный поиск
* быстрая сортировка
* бинарное дерево поиска
* очередь
* стек

## Возможности программы

Программа позволяет:

1. Показать среднюю температуру по месяцам
2. Найти самый тёплый день за год
3. Найти самый холодный день за год
4. Отсортировать месяцы по средней температуре
5. Найти дни теплее заданной температуры с помощью бинарного дерева поиска
6. Выполнить сглаживание температурного ряда методом скользящего среднего
7. Отменить последнее сглаживание с помощью стека
8. Сохранить текущий аналитический отчёт в файл

## Используемые алгоритмы и структуры данных

### Префиксные суммы

Префиксные суммы используются для быстрого подсчёта суммы температур за определённый интервал

Сначала программа строит массив накопленных сумм температур. После этого сумма температур за месяц находится как разность двух элементов массива префиксных сумм.

### Линейный поиск

Линейный поиск используется для нахождения самого тёплого и самого холодного дня

Программа последовательно проходит по списку всех дней и сравнивает температуры

### Быстрая сортировка

Для сортировки месяцев по средней температуре используется алгоритм быстрой сортировки

Сначала средние температуры по месяцам преобразуются в список пар:

```text
(месяц, средняя температура)
```

После этого список сортируется по значению средней температуры.

### Бинарное дерево поиска

Бинарное дерево поиска используется для поиска дней, когда температура была выше заданного пользователем значения

Ключом в дереве является температура, а значением является список записей с такой температурой. Это нужно для того, чтобы не терять дни с одинаковой температурой

При поиске программа выполняет рекурсивный обход дерева и выбирает только те узлы, где температура больше заданного порога.

### Очередь

Очередь используется для сглаживания температурного ряда методом скользящего среднего с окном 7 дней

В очереди хранятся последние 7 значений температуры. На их основе рассчитывается среднее значение.

### Стек

Стек используется для отмены последнего преобразования данных

Перед сглаживанием текущее состояние данных сохраняется в стек. Если пользователь выбирает отмену, программа достаёт предыдущее состояние из стека и восстанавливает его.

## Формат входных данных

Программа использует файл `weather.csv`.

Файл должен содержать 4 столбца:

```text
year,month,day,temperature
```

Пример данных:

```csv
year,month,day,temperature
2024,1,1,-5
2024,1,2,-3
2024,1,3,-7
```

Описание столбцов:

| Столбец       | Описание            |
| ------------- | ------------------- |
| `year`        | год                 |
| `month`       | месяц               |
| `day`         | день                |
| `temperature` | температура за день |

## Меню программы

После запуска пользователь видит консольное меню:

```text
СИСТЕМА АНАЛИЗА ПОГОДНЫХ ДАННЫХ
1. Показать среднюю температуру по месяцам
2. Найти самый тёплый день за год
3. Найти самый холодный день за год
4. Отсортировать месяцы по температуре
5. Найти дни теплее заданной температуры (BST)
6. Выполнить сглаживание ряда (Очередь 7 дней)
7. Отменить последнее сглаживание (Стек)
8. Сохранить текущий аналитический отчёт в файл
0. Выход из программы
```

Пользователь вводит номер пункта меню и получает результат в консоли.

## Пример работы

Пример поиска дней теплее заданной температуры:

```text
Выберите пункт меню (0-8): 5
Введите пороговую температуру: 20

Найдено дней теплее 20.0°C: 45
01.06.2024: 21°C
02.06.2024: 23°C
03.06.2024: 22°C
```

Пример сохранения отчёта:

```text
Выберите пункт меню (0-8): 8

Отчёт успешно сохранен в файл 'weather_report.txt'
```

## Сохранение отчёта

Программа может сохранить текущий аналитический отчёт в файл:

```text
weather_report.txt
```

В отчёт записываются:

* статус данных: исходные или сглаженные
* количество проанализированных дней
* самый тёплый день
* самый холодный день
* средняя температура по месяцам

## Ограничения проекта

Программа рассчитана на CSV-файл с погодными данными за один год.

Для корректной работы анализа по месяцам предполагается, что данные в CSV-файле расположены в хронологическом порядке.
